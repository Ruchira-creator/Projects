HR Analytics Dashboard
Business Intelligence Project — Power BI

📋 Project Overview
This project presents an interactive HR Analytics Dashboard built using Power BI.
The goal was to analyze employee attrition trends and provide actionable insights to HR teams for improving employee retention.

🎯 Dashboard Features
1️⃣ Key Metrics Summary
Total Employees: 1470

Attrition Count: 237

Attrition Rate: 16.12%

Average Age: 37 years

Average Salary: ~$6.5K

Average Years at Company: 7.0 years

2️⃣ Attrition Breakdown
By Education Field
E.g. Life Sciences, Medical, Marketing, Technical Degree, etc.

By Age Group
Groups: 18-25, 26-35, 36-45, 46-55, 55+.

By Salary Band
Bands: Upto 5k, 5k–10k, 10k–15k, 15k+.

By Gender
Male vs Female comparison.

By Job Role
Detailed role-based attrition analysis.
